# 行動支付模擬器
## 簡介
一開始先在APP內註冊會員，登入帳號後可建立交易，如新增收入與支出。<br>
帳號內的餘額若用以投資，則每日獲得些許收益，本利和亦可提取回帳號中。<br>
這些收入、支出、投資的交易紀錄皆列示於收支明細。<br>
最後在會員資料中可查看註冊時登記的資料，也能進行修改。<br><br>

## 功能
* 加入會員
* 忘記密碼
* 自動登入
* 新增收入
* 新增支出
* 投資服務
* 交易明細
* 會員資料<br><br>

## 資料儲存方式
* SQLite：舉凡會員資料、交易紀錄、投資歷史等，皆以資料庫來管理
* SharedPreference：僅用以做出自動登入之功能
